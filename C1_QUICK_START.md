## 5-Minute Setup (Basic C1)

### 1. Install Dependencies
```bash
cd aiapp
npm install @thesysai/genui-sdk @crayonai/react-ui zod zod-to-json-schema
```

### 2. Get Thesys API Key
- Sign up at https://thesys.dev
- Get your API key
- Add to `aiapp/.env.local`:
```bash
THESYS_API_KEY=your_key_here
```

## Troubleshooting

### "Tools not being called"
- Check system prompt mentions tools
- Verify TRADINGAGENTS_API is running
- Check tool definitions are correct

### "UI not rendering"
- Import C1 styles: `@crayonai/react-ui/styles/index.css`
- Check ThemeProvider wraps C1Chat
- Verify THESYS_API_KEY is set

### "Streaming not working"
- Check Content-Type header
- Verify async iteration
- Test with simple text first

### "Thread persistence not working"
- Check messageStore implementation
- Verify threadId is passed to API
- Check browser console for errors
